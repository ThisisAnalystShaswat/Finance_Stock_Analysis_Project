# Stocks Finance Analytics Project (Power BI)

###            Project Overview



This project showcases an end-to-end financial data analytics workflow using Power BI, focusing on ETL (extract, transform, load), data cleaning, data modeling, and analysis. The dataset includes various real-world data challenges, such as mixed date formats and invalid entries, to demonstrate practical problem-solving skills.



#### Tools \& Skills Used



* Power BI Desktop



* Power Query (M Language)



* DAX (for Measures \& Calculations)



* Excel for initial data review



* Financial and trading data analysis



#### ETL Process



1️⃣ Extract



* Imported multiple CSV files from a folder source for scalability.
* Loaded trading data and reference tables.



2️⃣ Transform (Data Cleaning)



* Fixed mixed date formats using multiple Date From Text conversions.
* Removed or replaced invalid values (NA, error) with null.
* Standardized text fields and handled duplicates.
* Applied business rules to fill missing dates with fallback values.



3️⃣ Load



* Loaded the cleaned data into the Power BI data model.
* Created a star schema with fact and dimension tables.





#### **Data Modeling**



* Built a fact table for trades and related dimension tables for traders, stocks, and exchanges.
* Established relationships to enable accurate DAX calculations and reporting.





#### Dashboard Features



* Interactive slicers for filtering by date, stock, and exchange.
* Key performance indicators (KPIs) for quick insights.
* Time series analysis of trade volumes and values.



#### How to Use This Project



* Clone or download the repository.



* Open the Stocks\_Finance\_Project.pbix file in Power BI Desktop.



* Review the Power Query Editor to see the cleaning steps.



* Explore the dashboard and visuals.
